Программирование на Python

Python

Python — это язык программирования высокого уровня, интерпретируемый, известный своей простотой и читаемостью. Создан Гвидо ван Россумом и впервые выпущен в 1991 году, Python стал одним из самых популярных языков программирования в мире.

Python поддерживает несколько парадигм программирования, включая структурированное, объектно-ориентированное, функциональное, императивное и аспектно-ориентированное программирование.

Почему учить Python?

  • Дружелюбен для новичков — Простая синтаксис, легко учиться
  • Многофункционален — Используется в веб-разработке, Data Science, ИИ и др.
  • Большое сообщество — Обширные библиотеки и фреймворки
  • Кроссплатформенный — Работает на Windows, macOS и Linux
  • Open source — Бесплатен для использования и распространения

Установка Python

Python доступен для всех основных операционных систем. Процесс установки прост и хорошо документирован.

Установка на Windows

  1. Скачайте установщик с официального сайта: python.org
  2. Запустите скачанный .exe файл
  3. Во время установки отметьте "Add Python to PATH"
  4. Нажмите "Install Now"
  5. Проверьте установку, открыв командную строку и введя python --version

Установка на macOS

  1. Скачайте установщик для macOS с официального сайта
  2. Откройте скачанный .pkg файл
  3. Следуйте инструкциям установщика
  4. Проверьте установку в Терминале командой python3 --version

Установка на Linux

Большинство дистрибутивов Linux поставляются с предустановленным Python. Для установки последней версии выполните:

Команды в терминале
# Для Ubuntu/Debian
sudo apt update
sudo apt install python3

# Для Fedora
sudo dnf install python3

# Проверка установки
python3 --version

Полезный совет

Используйте виртуальные окружения для изоляции зависимостей проекта. Создайте одно командой:

python -m venv myenv

Основы синтаксиса Python

Синтаксис Python разработан для читаемости с использованием значимых отступов и минимальной пунктуации.

Важно соблюдение отступов

Python использует отступы для определения блоков кода вместо фигурных скобок.

Пример
# Правильно
if 5 > 2:
    print("Пять больше двух!")

# Неправильно (выдаст ошибку)
if 5 > 2:
print("Пять больше двух!")

Комментарии

Комментарии начинаются с # и продолжаются до конца строки.

Пример
# Это однострочный комментарий

"""
Это многострочный комментарий.
Он может занимать несколько строк.
"""

print("Привет, мир!")  # Комментарий после кода

Попробуйте сами!

print("Привет, мир!")

result = 5 * 3
print("5 умножить на 3 равно:", result)

# Условное выражение
if result > 10:
    print("Результат больше 10!")
else:
    print("Результат 10 или меньше")

Переменные и типы данных

Переменные создаются при присвоении значения. Python автоматически определяет тип данных.

Основные типы данных

  • Целое число (int) — целые числа без дробной части
  • Число с плавающей точкой (float) — числа с дробной частью
  • Строка (str) — последовательность символов
  • Логический тип (bool) — True или False
  • Список (list) — упорядоченная изменяемая коллекция
  • Кортеж (tuple) — упорядоченная неизменяемая коллекция
  • Словарь (dict) — коллекция пар ключ-значение
Примеры
# Целые числа
age = 25

# Числа с плавающей точкой
temperature = 36.6

# Строки
name = "Иван"

# Логические значения
is_student = True

# Список
fruits = ["яблоко", "банан", "апельсин"]

# Кортеж
coordinates = (10.5, 20.3)

# Словарь
person = {
    "name": "Мария",
    "age": 30,
    "city": "Москва"
}

Попробуйте сами!

# Работа с числами
x = 10
y = 3.5
sum = x + y
print("Сумма:", sum)

# Работа со строками
greeting = "Привет, "
name = "Анна"
message = greeting + name
print(message)

# Работа со списками
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print("Список чисел:", numbers)

# Работа со словарями
book = {"title": "Война и мир", "author": "Толстой", "year": 1869}
print(f"Книга: {book['title']}, Автор: {book['author']}")

# Проверка типа
print("Тип x:", type(x))
print("Тип name:", type(name))
print("Тип numbers:", type(numbers))

Условные операторы

Условные операторы позволяют выполнять разные блоки кода в зависимости от условий.

Оператор if

Синтаксис
if условие:
    # выполнить, если условие истинно
elif другое_условие:
    # выполнить, если первое условие ложно, а это истинно
else:
    # выполнить, если все условия ложны

Попробуйте сами!

# Определение високосного года
year = 2024

if year % 4 == 0:
    if year % 100 == 0:

            print(f"{year} — високосный год")
        else:
            print(f"{year} — не високосный год")
    else:
        print(f"{year} — високосный год")
else:
    print(f"{year} — не високосный год")

Циклы в Python

Циклы позволяют многократно выполнять блок кода. В Python есть два типа циклов: for и while.

Цикл for

Используется для перебора последовательностей (списков, кортежей, строк и т.д.).

Примеры
# Перебор элементов списка
fruits = ["яблоко", "банан", "вишня"]
for fruit in fruits:
    print(fruit)

# Перебор символов строки
for char in "Python":
    print(char)

# Использование range()
for i in range(5):      # 0 до 4
    print(i)

for i in range(2, 6):   # 2 до 5
    print(i)

for i in range(0, 10, 2):  # 0, 2, 4, 6, 8
    print(i)

Цикл while

Выполняет блок кода, пока условие истинно.

Пример
# Счетчик
count = 0
while count < 5:
    print(count)
    count += 1  # то же самое, что count = count + 1

Попробуйте сами!

# Поиск простых чисел
print("Простые числа от 2 до 30:")
for num in range(2, 31):
    is_prime = True
    for i in range(2, num):
        if num % i == 0:
            is_prime = False
            break
    if is_prime:
        print(num, end=" ")

print("\n\nЧетные числа от 0 до 10:")
n = 0
while n <= 10:
    if n % 2 != 0:
        n += 1
        continue
    print(n, end=" ")
    n += 1

Попробуйте сами!

# Функция факториала
def factorial(n):
    """Вычислить факториал n"""
    if n == 0:
        return 1
    else:
        return n * factorial(n-1)

print("Факториал 5:", factorial(5))

# Функция Фибоначчи
def fibonacci(n):
    """Вернуть n-й число Фибоначчи"""
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a

print("Первые 10 чисел Фибоначчи:", end=" ")
for i in range(10):
    print(fibonacci(i), end=" ")
